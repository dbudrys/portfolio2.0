Data Analysis of 911 Calls in Baltimore  

Analysis provides insightful information about crime in the Baltimore area. Data set contains records from 2015 to August of 2017. Analysis includes a detailed explanation of the frequency of 911 calls, the most common reasons for calling, busiest weeks and days throughout the provided time span, and more. Project was developed using various Python data science libraries such as: Pandas, NumPy, Matplotlib, Seaborn, and others.
