Python Web Framework (Django) - Web Project  

Automative wbe app developed as a freelance project for car retail company. This is a dummy example of the project which showcases the design and technology used to complete the project. Automotive website developed using Django framework. Front-End developed using HTML5, CSS3 and , Materialize CSS library. This project containts Django Admin back-end interface to easily enter new car posting (access to back-end can be given upon request). Website is hosted using pythonanywhere.com hosting services.
