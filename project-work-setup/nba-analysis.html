NBA Analysis for the Correlation Between Points and Time  

Provided NBA analysis is try to answer the question: Does playing more minutes in the NBA guarantee scoring more points. Provided dataset includesevery players yearly statistic from 1950-2017, total of 24691 record. Results are generated using Apache Spark / Scala and its' Machine Learining library for K-Means. Final graph was developed using Python libraries such as: Matplotlib and Pandas
